## 1.0.13 - 2019-02-07
### Added
- Added support for `opts["uniquify"]` to set the uniquification mode via
  `m.compile`. Suppored values are `"UNIQUIFY", "WARN", "ERROR"`.

### Fixed
- Fixed bug in uniquification error mode.
